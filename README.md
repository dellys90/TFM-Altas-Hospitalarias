# TFM - Metodología de Visual Analytics aplicada al análisis de altas hospitalarias en España

## Descripción

Este repositorio contiene el material desarrollado para el Trabajo Fin de Máster (TFM) titulado:

**"Propuesta de una metodología de Visual Analytics para el análisis de altas hospitalarias en España (2010–2022)"**

El objetivo del trabajo es diseñar, desarrollar y validar una metodología para el análisis y la visualización interactiva de datos hospitalarios mediante Microsoft Power BI, facilitando la identificación de patrones y apoyando la toma de decisiones en el ámbito sanitario.

---

## Contenido del repositorio

### Datos

La carpeta **datos** contiene el conjunto de datos original utilizado para el desarrollo del trabajo.

- **crudo/**: datos originales empleados en el análisis.

### Dashboard

El archivo **Dashboard_Altas_Hospitalarias.pbix** contiene el dashboard desarrollado en Microsoft Power BI e incluye:

- Proceso de transformación de datos mediante Power Query.
- Modelo de datos.
- Medidas DAX.
- Visualizaciones interactivas.
- Dashboard final desarrollado para el TFM.

---

## Herramientas utilizadas

- Microsoft Power BI Desktop
- Power Query
- DAX
- Git
- GitHub

---

## Reproducibilidad

Para visualizar el dashboard es necesario disponer de **Microsoft Power BI Desktop**.

Al abrir el archivo **Dashboard_Altas_Hospitalarias.pbix**, es posible consultar el modelo de datos, las transformaciones realizadas y las visualizaciones desarrolladas durante el trabajo.
